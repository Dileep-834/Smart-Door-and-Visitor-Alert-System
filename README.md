# Smart Door and Visitor Alert System

## Project Overview

This project is an IoT-based smart home security system that detects visitors, counts people, and remotely controls a door using the Blynk mobile application.

## Features

- Motion Detection
- Visitor Counting
- Door Lock/Unlock
- Push Notifications
- Live Monitoring

## Hardware

- ESP8266 NodeMCU
- PIR Sensor
- IR Sensor
- Servo Motor

## Software

- Arduino IDE
- Blynk Platform

## Technologies

- IoT
- Embedded C
- ESP8266

## Applications

- Smart Homes
- Home Security
